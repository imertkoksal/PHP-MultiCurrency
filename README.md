# PHP MultiCurrency System

A lightweight PHP library for formatting and converting currency values into human readable text across multiple languages.

## Features

- Multi-currency support (USD, EUR, JPY, GBP, TRY, CHF, CNY, SEK)
- Multi-language number-to-words conversion
- Currency symbol formatting
- Fractional currency support (cent, kuruş, etc.)
- Simple PHP class structure
- No external dependencies

## Supported Languages

- English
- Turkish
- German
- Japanese

## Supported Currencies

- USD – US Dollar
- EUR – Euro
- JPY – Japanese Yen
- GBP – British Pound
- TRY – Turkish Lira
- CHF – Swiss Franc
- CNY – Chinese Yuan
- SEK – Swedish Krona
